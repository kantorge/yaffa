# YAFFA – Product Context (for AI Agents)

## Product Overview

YAFFA (Yet Another Free Financial Application) is a self-hosted personal finance web application focused on long-term financial planning and habit building.

Users host their own instance to retain full control over their financial data. No financial data is shared with third parties by default.

---

## Core Philosophy

- Financial awareness requires active participation
- Manual interaction with transactions is intentional and encouraged
- Automation should assist, not replace, user understanding
- Privacy and self-hosting are key differentiators

---

## Target Users

- Individuals managing personal finances
- Users interested in long-term planning and financial habits
- Privacy-conscious users preferring self-hosted solutions
- Technical users capable of self-hosting (secondary audience)

---

## Core Capabilities

- Multi-account and multi-currency financial tracking
- Transaction recording and categorization
- Intelligent suggestions to speed up data entry
- Transaction splitting for detailed analysis
- Budgeting and recurring transaction planning
- Forecasting based on scheduled data
- Investment tracking with price updates
- Optional AI-assisted document processing for transaction input

---

## Key Differentiators

- Self-hosted (data ownership)
- Emphasis on conscious financial tracking (not full automation)
- AI-assisted input (optional, user-controlled)
- Flexible categorization and transaction modeling

---

## Non-Goals / Explicit Limitations

- No automatic bank synchronization
- No native mobile application (responsive web only)
- Not intended as strict accounting software
- Precision handling for monetary values is currently limited (planned improvement)

---

## Feature Maturity Notes

- Some features are partially implemented or evolving
- Advanced reporting and onboarding are planned but incomplete
